# Housing Market Trends

A Flask web application that visualizes King County, WA housing market data through embedded Tableau dashboards. The project analyzes 21,609 synthetic housing records to uncover insights on sale prices, renovation impact, and market dynamics.

## Features

- **Home** — Hero landing page with key dataset statistics (21,609 records, $511K avg price, 31 features)
- **About** — Project overview and analysis objectives
- **Dashboard** — Embedded interactive Tableau dashboard for comprehensive house data analysis
- **Story** — Embedded Tableau story providing a guided narrative through key housing market insights

## Project Structure

```
├── app.py                  # Flask application and route definitions
├── generate_dataset.py     # Synthetic dataset generator (King County-style, 21,609 rows)
├── requirements.txt        # Python dependencies
├── Procfile                # Deployment process declaration
├── data/
│   └── housing_data.csv    # Generated housing dataset (31 features)
└── templates/
    └── index.html          # Single-page Jinja2 template with all views
```

## Dataset

The dataset is generated by `generate_dataset.py` and mimics real King County, WA housing sales data (May 2014 – May 2015). It contains **21,609 rows** and **31 features**, including:

| Feature | Description |
|---|---|
| `price` | Sale price (modeled from size, grade, location, waterfront) |
| `bedrooms` / `bathrooms` / `floors` | Physical structure |
| `sqft_living` / `sqft_lot` | Square footage |
| `grade` | Construction quality (1–13 scale) |
| `condition` | Property condition (1–5 scale) |
| `waterfront` / `view` | Premium property attributes |
| `yr_built` / `yr_renovated` | Age and renovation history |
| `lat` / `long` / `zipcode` | King County geographic location |
| `house_age` / `years_since_renovation` | Derived time features |
| `price_per_sqft` / `bed_bath_ratio` | Derived ratio features |
| `grade_category` / `condition_label` / `size_category` | Categorical labels |

## Getting Started

### Prerequisites

- Python 3.8+
- Tableau Desktop or Tableau Public (to open/edit the workbook)

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd Housing-Market-Trends
   ```

2. **Create and activate a virtual environment**
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # macOS / Linux
   source venv/bin/activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Generate the dataset** *(skip if `data/housing_data.csv` already exists)*
   ```bash
   python generate_dataset.py
   ```

5. **Run the application**
   ```bash
   python app.py
   ```
   The app will be available at `http://localhost:5000`.

## Deployment

The `Procfile` is configured for platforms like **Heroku** or **Render**:

```
web: python app.py
```

The app binds to the `PORT` environment variable automatically, falling back to port `5000` for local development.

## Tableau Integration

The Dashboard and Story pages embed publicly hosted Tableau views from **Tableau Public** (`GroupProject_17725154349810`). The embeds use the Tableau JavaScript API (`viz_v1.js`) and are fully interactive.

- **Dashboard** — `GroupProject_17725154349810/Dashboard1`
- **Story** — `GroupProject_17725154349810/Story1`

To use your own Tableau workbook, replace the embed code inside the `tableau-embed` divs in [templates/index.html](templates/index.html).
